# HeartClasses (Spigot 1.12.2) Made by Kharique Stevens For mastermanv2.eagler.host

A lightweight heart system plugin skeleton. It adds new hearts to the game with speicel ablitlitys!
It is fully open sorced. If u add things pls contact me at kwolfiscool@hotmail.com

## Build

```sh
mvn package
```

## Install

Drop the generated JAR from `target/` into your server's `plugins/` folder.

## Commands

- `/heartrecipes` - View heart recipes
- `/heartgui` - Open admin heart editor (requires `heartclasses.admin`)
- `/hearts` - View your current heart type/count
